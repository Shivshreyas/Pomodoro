# Flow - Focus Timer

A beautiful, minimalist Pomodoro timer with a glassmorphism design. Focus, track, and optimize your productivity with an intuitive interface. Select your timer mode (Focus, Short Break, or Long Break)

## Keyboard Shortcuts

- **Space**: Start/Pause timer
- **1**: Switch to Focus mode
- **2**: Switch to Short Break
- **3**: Switch to Long Break
- **R**: Reset current timer
- **S**: Skip to next session

## Customization

Access the Settings panel (gear icon) to:
- Adjust timer durations
- Enable/disable dark mode
- Choose alarm sounds
- Control volume
- Toggle tick sound feedback
- Auto-start breaks

## Browser Compatibility

Works on all modern browsers that support:
- CSS Custom Properties
- Backdrop Filter
- Web Audio API
- LocalStorage
- SVG

## Technical Stack

- Pure HTML5, CSS3, and Vanilla JavaScript
- No external dependencies
- Single file (`index.html`)
- ~1700 lines of optimized code
